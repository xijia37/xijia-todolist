Overview

51PM is a simple yet sophisticated project management tool. It is built on top of YII framework, an excellent PHP MVC framework.

It is assumed that you understand PHP and Mysql. If you need host the application in a hosting company, you need understand the hosting company's directory structure for you domain. After all, installing PHP applications is basically copying files from one location to another.

Installation

Installation is very straightforward as shown in the following steps:
  • Download 51pm.zip
  • Upload it to your web server. You can place it into the root directory or a subdir
  • Create a mysql database
  • Launch http://yourhost/{your install dir}/install.php
The installation process is 2-minutes process. In no time your 51PM system is ready for use!

I ran into a "internal server error".
Most likely, this is a problem related URL rewritting. Make sure the URL rewrite module is enabled in your Apache configuration. Then, open .htaccess at top level, notice these following lines: 
# otherwise forward it to index.php
RewriteRule . index.php

# the following is used when installed in root of a domain or subdomain
#RewriteRule ^.*$ /index.php [L]
